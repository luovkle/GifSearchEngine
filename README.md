# GifSearchEngine

## Requirements

* git
* node
* yarn
* [giphy](https://giphy.com/) api key

## Usage

```sh
git clone https://github.com/luovkle/GifSearchEngine
cd GifSearchEngine/frontend
echo "VITE_API_KEY=API_KEY" > .env
yarn install
yarn dev
```
